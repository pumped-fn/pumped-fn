# OKRA

This directory stores recoverable OKRA run state for repo-level work.

Tracked run records live under `.okra/runs/<run-id>/` and keep frame, tree, append-only ledgers,
flags, check-ins, drafts, move results, worker progress, terminal records, and referenced content
blobs. Generated status views, lock files, worker captures, and temporary files stay ignored.

```text
.okra/runs/invoice-triage-operational-20260706/
```
