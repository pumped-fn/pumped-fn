# CLI & Entry Points

## Goal

Launch codemod runs and expose the package entrypoints that wire transforms and
reporting together.

## Container Connection

This component ties the package together at the boundaries users touch first. Without
it, the codemod container would have transforms and reporting helpers but no supported
way to invoke them from a shell or import them from another tool.

## Dependencies

| Direction | What | From/To |
|-----------|------|---------|
| IN (uses) | Transform execution and collector access | c3-802 |
| IN (uses) | Markdown report generation | c3-803 |
| OUT (provides) | CLI invocation and package exports | c3-8 |
